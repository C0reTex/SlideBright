SlideBright
===========

Hexbright - An overengineered Flashlight

<h2>Annotation</h2>
<p>SlideBright is based on darcbright by Robert Quattlebaum <darco@deepdarc.com></p>

 <ol>

   <li>strobe mode</li>

 </ol>
